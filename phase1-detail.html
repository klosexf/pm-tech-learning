🎯 核心知识点总结

HTTP 请求流程

DNS解析 → 建立连接 → 发送请求 → 服务器处理 → 返回响应 → 渲染页面

前后端分工

前端负责界面展示和交互，后端负责业务逻辑和数据处理

API 三要素

接口地址(URL)、请求方法(GET/POST)、参数与返回值

数据流转路径

用户操作 → 前端 → API → 后端 → 数据库 → 返回结果
